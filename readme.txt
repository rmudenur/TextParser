
Program is to read the input text file and generate the classifiers for each sentences in the input text. 
This program uses openlnp tool and bin files to process the input text and generate the classifiers for the sentences.

Steps to execute this program. 

Provide the input text in the file - openlnp_model/input.txt file. 
Add stopwords if you have any into the file - openlnp_model/stopwords.txt
pom.xml file contains the dependency jars details. comppile using maven - "mvn eclipse:eclipse" to download the jars. 
run the main class - In the console you get the results
